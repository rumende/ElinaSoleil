Privacy Policy

Effective Date: July 10, 2025

This Privacy Policy explains how the Elina Soleil AI Instagram automation system handles data. This system is designed to automate content publishing and interaction through Meta's Instagram Graph API using AI-generated responses.

1. Information We Collect

The system does not collect or store personal information from Instagram users. It only interacts with public content (comments, messages) using API access granted by the account owner.

2. Use of Information

Any user interactions (e.g., responses to comments or messages) are generated by OpenAI's GPT-4 and not stored or reused. The system operates in real-time and does not log content.

3. Third-Party Services

The automation is powered by the following services:

  • n8n.io – for workflow automation
  • Meta (Instagram Graph API) – for accessing and managing Instagram content
  • OpenAI GPT-4 – for natural language generation of replies

4. Data Security

All access tokens are stored securely in the local n8n instance and are not exposed externally. No third-party analytics or tracking is used.
